# Part B: 边界匹配测试结果

测试目标：LLM 能否判断两个错误是否代表"相同"的失败模式

| Pair | 错误A | 错误B | LLM判断 | 预期 | 匹配? |
|------|-------|-------|---------|------|-------|
| 1 | sin(x²)求导遗漏内层 | ln(2x+1)求导遗漏内层 | **SAME** — 都是链式法则中系统性遗漏内层函数的导数，外层函数不同(sin vs ln)但失败模式完全一致 | true | YES |
| 2 | sin(x²)遗漏内层导数 | e^(3x)用幂函数规则 | **DIFFERENT** — 前者是步骤遗漏（知道要用链式法则但漏了一步），后者是规则混淆（根本用错了求导规则类型） | false | YES |
| 3 | 含参二次函数不分类讨论 | 二次函数只查端点不查顶点 | **DIFFERENT** — 前者的问题是"对称轴位置随参数变化需要讨论"，后者是"忘记在区间内找极值点"。虽然都涉及二次函数最值，但一个是参数分析遗漏，一个是极值点检查遗漏 | false | YES |
| 4 | r²=4误以为r=4 | 椭圆用c²=a²+b² | **DIFFERENT** — 前者是数值计算层面的混淆（平方和平方根），后者是公式记忆层面的混淆（椭圆vs双曲线）。虽然都可归类为"公式错误"，但失败的认知层次不同 | false | YES |
| 5 | sin(π+α)符号错误 | 不等式解集方向搞反 | **DIFFERENT** — 完全不同的知识领域。一个是三角函数象限符号，一个是二次不等式解集区间。无共同模式 | false | YES |
| 6 | 非标准均值不等式不会换元 | 含参二次函数不分类讨论 | **DIFFERENT** — 前者是代数变换技巧（换元将非标准形式转为标准形式），后者是参数讨论意识。虽然都属于"解题策略"层面，但具体的认知缺口完全不同 | false | YES |
| 7 | 不放回当放回(概率乘法) | 不放回取球用独立事件公式 | **SAME** — 本质是同一个错误的两种描述方式：不理解不放回抽样改变了样本空间，因此错误地使用独立事件的概率乘法 | true | YES |
| 8 | 立体几何未验证垂直 | 二次函数只查端点 | **DIFFERENT** — 虽然在元认知层面都有"遗漏检查步骤"的共性，但在具体知识点层面完全无关（一个是空间几何关系验证，一个是函数极值判断）。作为知识边界来说，应该分开对待 | false | YES |
| 9 | e^(3x)用幂函数降次 | 椭圆用双曲线c²公式 | **DIFFERENT** — 都是"公式混淆"，但跨知识领域（导数vs解析几何）。在教学上需要分别处理，不能因为都是"记错公式"就归为同一边界 | false | YES |
| 10 | sin(x²)遗漏内层 | ln(2x+1)遗漏内层 | **SAME** — 和 Pair 1 相同，链式法则内层导数的系统性遗漏 | true | YES |

## 统计

- 总计：10 pairs
- 正确判断 SAME：3/3
- 正确判断 DIFFERENT：7/7
- **准确率：10/10 = 100%**

## 关键观察

1. LLM 能准确区分"表面相似但本质不同"的错误（Pair 3：都是二次函数，但失败模式不同）
2. LLM 能准确识别"不同描述但本质相同"的错误（Pair 7：同一个条件概率错误的两种表述）
3. LLM 不会因为"都是公式错误"就简单归类为同一模式（Pair 4, 9）
4. LLM 能在给出判断的同时解释理由，这些理由在教学场景中直接可用
5. 对于跨知识领域的"结构相似"错误（如 Pair 8 都是"遗漏检查"），LLM 正确地判断为不同边界，因为教学上需要分别处理
